# myportfolio

This is a ready-to-run Vite + React + Tailwind project prepared for Gayathri.

How to use:
1. Extract the ZIP.
2. Open a terminal inside the `myportfolio` folder.
3. Run `npm install`.
4. Run `npm run dev`.
5. Open http://localhost:5173 in your browser.

Notes:
- Tailwind and PostCSS are already configured.
- Theme toggle (dark/light) persists using localStorage.
